Помощник для умного прибора учёта электроэнергии

Проверка ИПУ Расшифровка кодов ошибок ИПУ Расшифровка модели ИПУ Расчёт сечения кабеля

Выполнено без использования api, backend. Работает без доступа в интернет. Подключен service-worker: можно добавить на главный экран в качестве приложения

  • Форма для проверки корректности учёта электроэнергии прибором учёта
  • Форма для расшифровки кода ошибки прибора учёта электроэнергии, которая отображается на его экране, в текстовый вид. Расшифровка кода ошибки производится в соответствии с актуальным руководством по эксплуатации соответствующего прибора учёта
  • Форма для расшифровки обозначения модели прибора учёта
  • Расчёт сечения кабеля по введённым нагрузкам (локальный расчёт, без обращения к серверу)

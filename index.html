Помощник для умного прибора учёта электроэнергии

Выполнено без использования api, backend. Работает без доступа в интернет. Подключен service-worker: можно добавить на главный экран в качестве приложения

  • 1 - Форма для проверки корректности учёта электроэнергии прибором учёта.

  • 2 - Форма для расшифровки кода ошибки прибора учёта электроэнергии, которая отображается на его экране, в текстовый вид. Расшифровка кода ошибки производится в соответствии с актуальным руководством по эксплуатации соответствующего прибора учёта.

  • 3 - Форма для расшифровки обозначения модели прибора учёта.
